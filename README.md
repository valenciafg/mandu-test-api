# Mandu-test-API
Api de conexion con BD de divisiones bajo MySQL

## Instalar paquetes
### `composer install`

## Cargar tablas
### `php artisan migrate`

## Cargar data de prueba
### `php artisan db:seed`

## Ejecutar en local
### `php artisan serve`
